# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-02

### Added
- Initial open source release of Lightroom Ratio Creator
- Support for popular social media aspect ratios (Instagram, TikTok, YouTube)
- Standard print format ratios (3:2, 4:3, 5:4, 8:10)
- Smart orientation detection for optimal ratio application
- Automatic centered cropping functionality
- Virtual copy creation (non-destructive workflow)
- Collection management for organized virtual copies
- MIT License for open source distribution

### Features
- **Social Media Ratios**:
  - Instagram Square (1:1)
  - Instagram Portrait (4:5)
  - Instagram Stories/TikTok (9:16)
  - YouTube/Facebook (16:9)
- **Print Ratios**:
  - 35mm Film (3:2)
  - Traditional (4:3)
  - Large Format (5:4)
  - Portrait Print (4:5)
- **Smart Features**:
  - Orientation-aware ratio selection
  - Centered crop positioning
  - Batch processing support

### Technical
- Compatible with Adobe Lightroom Classic 10.0+
- Built with Lightroom SDK 11.0
- Optimized virtual copy creation
- Memory-efficient processing
