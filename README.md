3. **Open Adobe Lightroom Classic**

5. Click **Add** and select the `.lrplugin` folder
6. Click **Done**

## ğŸ“– Usage

1. **Select Photos** in Lightroom Library or Grid view
2. Go to **Library > Plug-in Extras > Ratio Creator**
3. **Choose Aspect Ratios** you want to create
4. **Configure Options**:
   - Smart orientation detection

   - Naming preferences
5. Click **Create** to generate virtual copies
